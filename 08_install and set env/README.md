## Before Starting 
* Install node
* Install visual studio code (as text editor)

## Set Angular Env  
* update node version:
```
sudo npm cache clean -f
sudo npm install -g n
sudo n stable
```
in ubuntu run also:
```bash
sudo apt-get install --reinstall nodejs-legacy     # fix /usr/bin/node
```
* check node version:
```
node --version
```
* update npm version:
```
npm install -g npm 
```
* check npm version:
```
npm --version
```
* install angular
```
npm i -g @angular/cli
```
* check angular version:
```
ng --version
```
